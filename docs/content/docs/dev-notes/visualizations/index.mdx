this document catalogs the official visualization styles for `spicetify-viz`. each visualizer is a modular, highly customizable engine for creative expression, designed to transform sound into a compelling visual narrative.

<Callout title="the visualizer pipeline">
  the system functions by taking real-time audio analysis data from the spicetify player and feeding it into a selected visualizer module. the visualizer then uses this data to render graphics to a high-performance html5 canvas element overlaid on the spotify ui.
</Callout>
