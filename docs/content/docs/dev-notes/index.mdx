### project vision

to engineer the definitive real-time audio visualizer for the spotify desktop client. this project serves as a public benchmark for excellence in modern, performant web technologies and is a showcase of professional software engineering practices. it is a creative playground for developers and a testament to the power of high-quality, open-source extensions.

### who this is for

this documentation is for developers, designers, and enthusiasts who wish to understand, contribute to, or learn from the `spicetify-viz` project. whether you're looking to build your own unique visualizer, resolve a complex performance issue, or simply study a modern graphics-intensive application, these documents are your authoritative guide. a strong working knowledge of typescript, react, and contemporary web development practices is assumed.

this is the official engineering documentation for `spicetify-viz`. it covers architecture, development, and contributions.
