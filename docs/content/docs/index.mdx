## Quick Start

Get up and running with `spicetify-viz` in a few simple steps. For more detailed instructions and configuration options, head over to the [Usage guide](./usage).

<Steps>
### Prerequisites
You'll need to have [Spicetify](https://spicetify.app/docs/getting-started) installed and working.

### Installation via Marketplace
The easiest way to install `spicetify-viz` is through the official Spicetify Marketplace, which handles automatic updates.

1.  Open Spotify and navigate to the `Marketplace` tab on the left sidebar.
2.  Search for `spicetify-viz`.
3.  Click `Install`.
4.  Navigate to the visualizer by clicking the new `spicetify-viz` icon in your sidebar and enjoy!
</Steps>

For alternative installation methods, see the [Usage guide](./usage).
